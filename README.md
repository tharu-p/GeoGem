# GeoGem
GeoGem is a simple business finder application using Flask, Python and HTML/CSS.

## ✨ Key Features
- **AI-Powered Summaries:** Instant insights into local businesses.
- **Gamified Rewards:** Earn points and badges for supporting local.
- **Dynamic Discovery:** Filter by rating, category, and neighborhood.

## 🚀 How to Run
1. Clone the repo: `git clone [your-link]`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `app.py`
